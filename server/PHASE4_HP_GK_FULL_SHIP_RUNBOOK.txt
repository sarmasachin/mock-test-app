3. DB (one-time, if timer still OFF):

     node scripts/phase0HpGkCycleSetup.js        # audit
     node scripts/phase0HpGkCycleSetup.js --apply  # enable timer only
4. APP: install PHASE4 APK on test device, then roll out

     b. At/after 9 PM → Start Test
