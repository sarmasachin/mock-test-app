Phase 0 — Remove fixed 20s/30min start lock; admin schedule-timer checkbox
============================================================================

WHAT CHANGED
------------
- REMOVED: startSeriesLockSeconds (20 sec) and startSeriesActiveWindowMinutes (30 min)
- ADDED: startSeriesScheduleTimerEnabled (boolean, default false)

BEHAVIOUR
---------
Timer OFF (default):
  - User applies → Start button available immediately
  - Re-apply after test cycle is governed by server cycle/republish (unchanged)
  - No artificial 20 sec countdown, no 30 min expiry window

Timer ON:
  - User applies → locked until exam date + slot (if set on test)
  - Expiry uses lateJoinMinutes from test config, else 24h from scheduled start
  - No fixed 20 sec / 30 min values

ADMIN
-----
Home Content → checkbox: "Schedule timer (lock start until exam date/slot)"
  - Unchecked = instant start after apply (recommended for practice/cycle tests)
  - Checked = date-based lock for scheduled exams
