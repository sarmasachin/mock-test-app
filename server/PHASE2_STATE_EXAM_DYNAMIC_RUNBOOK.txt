ADMIN HOW-TO
------------
1. Open admin → **State Exam Manager**
2. **State:** e.g. Himachal Pradesh
3. **Section:** e.g. शिक्षक भर्ती (teaching)
4. **Test link:**
   - *Use existing test* — pick from All Tests list
   - *Create new test* — tick "Create draft test" (add questions later)
5. **Exam name:** e.g. HP Math Teacher (this becomes Level 3 + circle label)
6. **Featured** ⭐ if important (shows first in section — Phase 3 UI)
7. **Order:** 1 = top among non-featured
8. Click **Save state exam circle**

Checks panel shows errors before save. Preview shows iconKey auto (e.g. hp:hp-math-teacher).
